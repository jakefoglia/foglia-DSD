To load and run one of the programs into the memory chip:

Open the project normally.
In the computer component (shown by default), select the adjust tool and click on the "MEM" component.
Click on the "Browse" button in the "Read File" section.
Find and select one of the "*.bin" files inside the "programs" folder.
Close the properties window.

Run the circuit.
Set the clock toggle (lower left) to 1.
Set the "Rst" toggle to 1, and set it back to 0 when the "Rdy" signal is 1.
Set the "Exe" toggle to 1 to start the program's execution.

Look inside the running processor to watch it work.
